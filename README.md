## 专辑介绍

《甜甜圈 (Sweet Circle)》是AI歌手"爱乐儿"倾情演绎的一张充满温暖与甜蜜的概念专辑。整张专辑以"圆满"与"甜蜜"为核心意象，通过16首风格各异的歌曲，勾勒出一幅关于爱、梦想与希望的音乐画卷。专辑融合了中国风的诗意浪漫与现代流行的律动节奏，从《甜甜圈》的纯真喜悦到《故人颜》的深情回望，从《惊鸿一念》的刹那惊艳到《逆光》的勇敢前行，再到《人间星火》的温暖传递，每一首歌曲都是情感的一个切面。星光主题的三首变奏曲（星光落在手心、掌心、指尖）以细腻的视角变化，展现了光芒与情感的不同层次；而《星火坠落》的两个版本则以截然不同的编曲风格，诠释了同一段旋律的恢宏与柔情。专辑最后以《月下听风吟》《云间谣》《云间月》的诗意意境，以及《追光的女孩》的励志力量作结，完成了一次从甜蜜到温暖、从星月到追光的完整情感循环。
